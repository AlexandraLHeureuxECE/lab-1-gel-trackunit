# Tic-Tac-Toe Game

A web-based Tic-Tac-Toe game built with plain HTML, CSS, and JavaScript (no frameworks).

## Features

- 3x3 game grid with two-player turn-based gameplay
- Win and draw detection with visual feedback
- Current player display with colored indicators
- Restart button (disabled until first move is made)
- Light/Dark theme toggle with system preference detection
- Hover preview showing where your mark will be placed
- Animated win celebration and draw notifications
- Responsive and modern UI design

## How to Play

1. Open `index.html` in any web browser
2. Player X goes first - click any cell to place your mark
3. Players alternate turns until someone wins or it's a draw
4. Click "Restart Game" to play again
